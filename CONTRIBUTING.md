
## Hoe kun je bijdragen / dingen toevoegen aan dit project:
### Optie 1: (Als contributor):
```
Stap 1
Maak een branch in de repository met als naam de feature die je toe wilt voegen / de aanpassing die je wilt maken

Stap 2
Voeg dingen toe of pas aan
Graag uitkijken met bestaande code verwijderen of even in overleg doen, want dat zorgt vaak voor conflicten bij het mergen.

Stap 3
Als jou code af is, doe een merge request om de nieuwe code in je branch toe te voegen aan de master branch

Stap 4
Merge request moet nog even geaccepteerd worden. Dit kan gedaan worden door de eigenaar en door de contributors.

```


### Optie 2:
```
Stap 1
Fork de repository naar je eigen github account

Stap 2
Voeg dingen toe of pas aan
Graag uitkijken met bestaande code verwijderen of even in overleg doen, want dat zorgt vaak voor conflicten bij het mergen.

Stap 3
Doe een pull request in mijn repository, om jou code en documenten toe te voegen aan deze repository.
Dit kan hier:
https://github.com/sKyW1per/ransomware-data/pulls

Stap 4
Als je een pull request hebt gedaan dan krijgen de owners en de contributors een melding, en kan de pull request accepteren of weigeren.


Een aanvraag doen voor een wijziging kan ook:
Ga naar 
https://github.com/sKyW1per/ransomware-data/issues
En vul iets in.
Maar als het veel werk is dan garandeer ik niet dat ik het voor je uitvoer :)
